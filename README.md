# Personalkampagne PAVODENS – Übersicht

Komplettpaket zur Mitarbeitergewinnung für **PAVODENS – Moderne Zahnmedizin**, Bobingen bei Augsburg.
Gesucht: **ZFA** und **ZMP** (m/w/d) · Zahntechniker als pausierte Talent-Pipeline.

## Dateien

| Datei | Was es ist |
|-------|-----------|
| **`index.html`** | Die fertige Recruiting-Landingpage. Im Browser öffnen oder hosten. Eine klare Conversion-Aktion („In 2 Minuten bewerben"), mobil-optimiert, warmes Praxis-Design. |
| **`Google-Ads-Kampagnenplan.md`** | Der vollständige Kampagnenplan: Strategie, Struktur, Geo-Targeting, Budget, Keywords, Anzeigen, Erweiterungen, Conversion-Tracking, Import-Anleitung, Optimierungsfahrplan + komplette Keyword-/Anzeigenlisten im Anhang. |
| **`pavodens-google-ads-import.csv`** | Importfertige Datei für den **Google Ads Editor** (3 Kampagnen, 7 Anzeigengruppen, 134 Keywords, 43 negative Keywords, 7 Responsive Search Ads). |
| `build_csv.py` | Referenz-/Generatorskript, mit dem die CSV erzeugt und auf Zeichenlimits geprüft wurde (0 Verstöße). |
| `.claude/launch.json` | Konfiguration für die lokale Landingpage-Vorschau. |

## Was du jetzt tun solltest – in 3 Schritten

1. **Landingpage hosten.** `index.html` auf einer URL veröffentlichen (z. B. `karriere.pavodens.de`). Bis dahin zeigen die Anzeigen auf das bestehende, funktionierende Bewerbungsportal.
2. **CSV importieren.** Im Google Ads Editor: *Konto → Importieren → Aus Datei…* → `pavodens-google-ads-import.csv`. Danach Geo-Targeting („Präsenz"-Option!), Sprache, Conversion-Tracking und Anzeigenerweiterungen ergänzen (alles Schritt für Schritt im Kampagnenplan, Abschnitte 3, 7, 8 und 9).
3. **Live gehen & optimieren.** ZFA + ZMP aktivieren (Zahntechniker bleibt pausiert). In den ersten 2 Wochen täglich den Suchbegriffbericht prüfen; nach 15–30 Bewerbungen auf „Conversions maximieren" umstellen.

## Wichtig

- Alle Anzeigentexte halten die Google-Zeichenlimits ein (programmatisch geprüft).
- Vor dem Schalten von Tracking-Tags: **Datenschutzerklärung & Cookie-Consent** aktualisieren.
- Die Kampagne arbeitet ausschließlich mit **echten** Vorteilen von der Praxis-Website – keine erfundenen Zahlen oder Bewertungen.
